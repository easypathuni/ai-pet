# AI Pet Assistant

A cute AI pet assistant that reads the current screen, reacts with playful animations, and helps users in the active context.

## Features

- Reads and understands the current screen context
- Provides helpful suggestions based on what is on screen
- Shows cute animations and interactive reactions
- Acts like a friendly on-screen companion
- Designed to be lightweight, useful, and fun

## About

AI Pet Assistant is an interactive desktop companion designed to make screen-based help feel more natural and enjoyable. Instead of a standard assistant panel, it appears as a cute virtual pet that reacts, moves, and supports the user on the current screen.

## Goals

- Make assistance more engaging and human-friendly
- Provide contextual help without feeling intrusive
- Create a fun pet-like experience with useful AI behavior
